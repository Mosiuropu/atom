# Language Molecular Bio

Syntax highlighting and snippets for molecular biology and bioinformatics file formats in Atom. Part of [Mosiur's Atom Fork](https://github.com/Mosiuropu/atom).

## Supported File Formats

| Format | Extension | Description |
|--------|-----------|-------------|
| FASTA | `.fasta`, `.fa`, `.fna`, `.faa` | Sequence files (nucleotide & protein) |
| FASTQ | `.fastq`, `.fq` | Sequencing quality scores |
| BED | `.bed` | Genomic interval data |
| GFF3 | `.gff`, `.gff3`, `.gtf` | Genome annotation |
| VCF | `.vcf` | Variant Call Format |
| GenBank | `.gb`, `.gbk`, `.genbank` | Annotated sequence records |

## Features

- Syntax highlighting for all major bioinformatics formats
- Color-coded nucleotide and protein sequences
- Metadata and comment highlighting
- Attribute recognition for annotation files
